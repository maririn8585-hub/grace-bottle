GRACE Bottle Firebase版

1. ZIPをMacに保存します。
2. ZIPをダブルクリックして展開します。
3. 中の index.html / firestore.rules / README.txt の3つをGitHubのアップロード画面へまとめてドラッグします。
4. 画面下の「Commit changes」を押します。

Firestore設定:
Firebase Console → Firestore Database → ルール
firestore.rules の中身を貼り付けて「公開」を押してください。

注意:
このルールは動作確認用です。URLを知っている人が読み書きできるため、実運用前にログイン機能を追加して安全な設定に変更してください。
